Plant Database API Privacy Policy

Effective Date: January 15, 2024
Last Updated: January 15, 2024

1. Introduction

This Privacy Policy describes how the Plant Database API ("we," "our," or "us") collects, uses, and protects information when you use our plant database management service through ChatGPT Actions or direct API access.

2. Information We Collect

2.1 Plant Data

We collect and store the following types of plant information that you provide:

  • Plant Identification: Plant names, descriptions, and locations in your garden
  • Care Information: Watering needs, light requirements, soil preferences, fertilizing schedules
  • Growing Conditions: Frost tolerance, spacing requirements, pruning instructions
  • Seasonal Care: Winterizing instructions, mulching needs
  • Photos: Plant photo URLs (we store URLs only, not the actual images)
  • Care Notes: Additional notes and observations about your plants

2.2 Technical Data

We automatically collect certain technical information:

  • API Usage Logs: Timestamps, IP addresses, and request details for security and debugging
  • Error Logs: Technical error information to improve service reliability
  • Rate Limiting Data: Request frequency to enforce usage limits

2.3 ChatGPT Integration Data

When accessed through ChatGPT Actions:

  • We receive requests from ChatGPT on your behalf
  • We do not collect ChatGPT conversation data
  • We do not store ChatGPT user identifiers

3. How We Use Your Information

3.1 Primary Uses

  • Plant Management: Store, retrieve, and update your plant database
  • Search Functionality: Enable searching through your plant collection
  • Care Reminders: Provide access to care instructions and schedules
  • Data Organization: Organize plants by location, care needs, and other attributes

3.2 Technical Uses

  • Service Operation: Maintain and improve API functionality
  • Security: Monitor for unauthorized access and prevent abuse
  • Debugging: Troubleshoot technical issues and improve reliability
  • Rate Limiting: Ensure fair usage and prevent system overload

4. Data Storage and Security

4.1 Data Storage

  • Plant data is stored in Google Sheets via Google Sheets API
  • API logs are stored on Render.com infrastructure
  • We use industry-standard encryption for data transmission
  • All API communications use HTTPS encryption

4.2 Security Measures

  • API Key Authentication: Secure access controls for write operations
  • Rate Limiting: Protection against abuse and excessive usage
  • Audit Logging: Complete logs of all data modifications
  • Access Controls: Limited access to authorized users only

5. Data Sharing and Disclosure

5.1 No Third-Party Sharing

We do not sell, trade, or otherwise transfer your plant data to third parties, except as described below:

5.2 Service Providers

  • Google Sheets: For data storage and management
  • Render.com: For API hosting and infrastructure
  • ChatGPT/OpenAI: When you use ChatGPT Actions (they facilitate requests but don't store your plant data)

5.3 Legal Requirements

We may disclose information if required by law, court order, or to protect our rights and safety.

6. Your Rights and Controls

6.1 Data Access

  • You can view all your plant data through the API
  • You can export your data at any time
  • You have full control over your plant database

6.2 Data Modification

  • You can update any plant information at any time
  • You can add new plants or remove existing ones
  • You can modify care instructions and notes

6.3 Data Deletion

  • You can delete individual plants from your database
  • You can request complete data deletion by contacting us
  • Audit logs may be retained for security purposes

7. Data Retention

  • Plant Data: Retained until you delete it or request account closure
  • API Logs: Retained for 90 days for security and debugging purposes
  • Error Logs: Retained for 30 days for troubleshooting

8. International Data Transfers

Your data may be stored and processed in the United States or other countries where our service providers operate. We ensure appropriate safeguards are in place for international data transfers.

9. Children's Privacy

Our service is not directed to children under 13. We do not knowingly collect personal information from children under 13. If you believe a child has provided us with personal information, please contact us immediately.

10. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify users of significant changes by updating the "Last Updated" date above. Continued use of the service constitutes acceptance of the updated Privacy Policy.

11. Contact Information

Data Controller: Plant Database API

Contact: For privacy-related questions, requests, or concerns:

  • Submit issues through our GitHub repository
  • Contact via the API documentation channels
  • For data deletion requests, include your API key for verification

Response Time: We aim to respond to privacy requests within 30 days.

12. Legal Basis for Processing (GDPR)

For users in the European Union, our legal basis for processing your data is:

  • Legitimate Interest: To provide plant database management services
  • Consent: When you voluntarily provide plant information
  • Contract Performance: To deliver the API services you request

This privacy policy is designed to be comprehensive and transparent about our data practices. If you have questions about any part of this policy, please contact us.
